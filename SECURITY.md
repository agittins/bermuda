# Security Policy

## Supported versions

Only the latest release on the `main` branch is supported with security fixes.

## Reporting a vulnerability

**Do NOT open a public issue.** Instead, report vulnerabilities privately via:

- GitHub private vulnerability report: [Open a security advisory](https://github.com/foXaCe/bermuda/security/advisories/new)

Target response time: 7 days.

Please include:
- A description of the vulnerability and its impact
- Steps to reproduce
- Affected versions (if known)
- Suggested fix (optional)

## Disclosure

Once a fix is released, a security advisory is published on the [security advisories page](https://github.com/foXaCe/bermuda/security/advisories).
